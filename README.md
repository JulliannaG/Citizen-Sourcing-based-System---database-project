# Citizen-Sourcing-based-System for Smart City---database-project

## About the Project
This is a smart city database project for Pathumthani Province. The main idea is a system where citizens can easily report local issues (like broken streetlights or road damage) directly to the right government offices.

## Main Features
* **Reporting:** Citizens can report problems, add the location, and attach media (photos/videos).
* **Routing:** Issues are categorized and assigned to the correct government department.
* **Feedback:** Users can track the status and leave a rating/feedback once the issue is resolved.

## 3 Database Implementations
The main technical goal of this project was to design a conceptual data model and then implement the exact same system using three different database technologies. 

The database was implemented in:
1. **MySQL** (Relational)
2. **MongoDB** (Document-oriented)
3. **Neo4j** (Graph)
